## Image Naming Convention

- **Hero:** `hero-{descriptor}.jpg` (e.g., hero-main, hero-secondary)
- **Projects:** `project-{number}-{category}.jpg` (e.g., project-07-audit, project-15-commercial)
- **Gallery:** `{type}-{descriptor}-{number}.jpg` (e.g., site-inspection-01, equipment-upv)
